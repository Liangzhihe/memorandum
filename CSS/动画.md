## 动画

    div {
        animation: mymove 1s;
        animation-iteration-count: infinite;
    }
        
    @keyframes mymove {
        0% {
            transform: translateY(0px);
        }
        50% {
            transform: translateY(8px);
        }
        100% {
            transform: translateY(0px);
        }
    }

    // translateY(param)函数表示在页面垂直移动元素
    // translateX(param)函数表示在页面水平移动元素
